Mission statement

This is a testing server for this codebase:

github.com/smiley1983/aiweb

There are two purposes to running this server now.

  • To improve the codebase, discovering issues and bugs and prompting discussion about what such a server system should do.
  • To provide a public space where people can see their bots playing against others.

This server is not part of a planned contest: this is alpha testing of in-development code. There may be security issues, the database might get wiped, it's even plausible that someone might steal your precious source code from the server.

All that said, you are encouraged to upload bots here if you're interested in the AI programming games offered. There's value in writing bots for those games and running them on your own computer, but without a public server like this, nobody will know if you write the best bot in the world. The games used in the AI challenges were excellent, both for beginners and advanced programmers. This server is offered partly to give life to those games, which still have much potential for further AI development.

The future

In the short term, there are plans to:

  • update the TCP server system
  • get all of the programming languages running
  • improve the website
  • sandbox the compilation step and address other possible security issues.
  • get PlanetWars running (and perhaps Rock, Paper, Scissors as well)

In the long run, it would be great to see another contest like the aichallenge, possibly even running a derivative of this code. Perhaps you're interested in running such a thing?

In the meantime, if you're looking for an active contest, have a look at The AI Games. It looks like things are happening there.
